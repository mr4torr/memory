## Instalar

Instalar QMD globalmente

```
npm install -g @tobilu/qmd
# or
bun install -g @tobilu/qmd

```

Definir a pasta de indexação

```
qmd collection add ./Z01\ -\ AI\ Obsidian/Wiki --name wiki
```

Criar os alias para os IDE's e TUI's

```
./setup.sh
```
